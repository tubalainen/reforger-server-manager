## What's new in 0.42.0

### Mods Overview (#131)
A new **Mods Overview** tab gives you one persistent, template-independent list of every
mod that has ever been baked into a server template — so you finally have a single place to
see and reuse your mods instead of hunting through each template.

- **Tree of mods and dependencies.** Every mod you've added is a root, expanded with its
  dependencies resolved live from the Workshop. A mod that's both standalone and a
  dependency of another shows up in both places, and dependency-only mods are labelled as
  such.
- **Baked & downloaded, highlighted.** Each mod shows a green badge for every server it's
  baked into, with the version that server's template configures.
- **Once added, always listed.** A mod stays on the overview even after the template that
  carried it is deleted — flagged **unused** until you add it somewhere again.
- **Add to a template in one click.** Tick mods at any level in the tree, pick a template,
  and they're merged straight into it (its dependencies download at runtime, exactly like
  adding a mod in the wizard). The template's change log records it.
- **Persist and prune.** Pin any mod as **persist** so it's never removed, and use
  **Clear &amp; rescan** to drop the mods no template uses any more and re-sync the list to
  your templates. Persisted mods always survive.

**Upgrade:** the manager updates itself on start, or pull
`ghcr.io/tubalainen/reforger-server-manager:latest`.
