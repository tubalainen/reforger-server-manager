## What's new in 0.43.0

### Mods Overview now shows each mod's type (#131)
The Mods Overview tab was hard to read when every entry looked the same — you couldn't
tell a scenario from an ordinary mod. Each entry is now labelled with its **Workshop
type**:

- A coloured pill marks every mod as **Scenario**, **Terrain** or **Mod** (addon),
  resolved live from the Arma Reforger Workshop, so scenarios stand out at a glance.
- A **+ scenario** hint flags a plain mod that also publishes its own scenario(s).
- The mod's **category tags** from the Workshop (Vehicles, Weapons, Sound, …) show as
  small chips on each row.

Types load with the dependency tree, so they appear as soon as the Workshop responds and
are simply omitted (rather than guessed) when it's unreachable — a mod known to publish
scenarios is still flagged from saved template data even then.

**Upgrade:** the manager updates itself on start, or pull
`ghcr.io/tubalainen/reforger-server-manager:latest`.
