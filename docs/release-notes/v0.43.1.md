## What's new in 0.43.1

### Mods Overview: ticking a mod selects its whole dependency subtree (#131)
In the Mods Overview tree, selecting a top-level mod now automatically selects everything
beneath it — its dependencies, and their dependencies. Clearing a parent clears the
subtree the same way. Adding a mod and all of its dependencies to a template is now a
single click instead of ticking each node by hand.

**Upgrade:** the manager updates itself on start, or pull
`ghcr.io/tubalainen/reforger-server-manager:latest`.
