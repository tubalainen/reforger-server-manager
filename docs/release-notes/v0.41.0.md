## What's new in 0.41.0

### Copy a server template (#129)
The Templates page now has a **Copy** button on every template. It duplicates the
template — scenario, mods, settings, launch parameters and any custom config keys —
into a new "Copy of…" one and drops you straight into its editor. Start a variant
from a known-good base instead of rebuilding from scratch; the copy is fully
independent, so editing or deleting it never touches the original. Each copy also
gets its own change log.

**Upgrade:** the manager updates itself on start, or pull
`ghcr.io/tubalainen/reforger-server-manager:latest`.

Full history: https://github.com/tubalainen/reforger-server-manager/compare/v0.40.0...v0.41.0
