## What's new in 0.43.5

### Mods Overview: clearer loading, and a collapsed tree you expand yourself (#131)
Two refinements to the Mods Overview tab:

- **A visible "loading" state while the tree builds.** Resolving each mod's dependencies
  hits the Workshop, which can take a while — the page no longer looks flat and finished
  during that time. A spinner and *"Building the dependency tree from the Workshop — this
  can take a moment…"* now show while it resolves.
- **The tree starts collapsed.** Instead of everything being expanded by default, each mod
  with dependencies now starts closed and you expand the branches you care about with the
  ▸ toggle. **Expand all** / **Collapse all** buttons are there when you want the whole
  tree at once.

**Upgrade:** the manager updates itself on start, or pull
`ghcr.io/tubalainen/reforger-server-manager:latest`.
